### Conflict Between Team Members

```
┌─────────────────────────────────────────────────────────────────┐
│              CONFLICT RESOLUTION                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  John and Amy disagree on design direction...                  │
│                                                                 │
│  John → Nami: "Design ควรเป็น minimal"                          │
│  Amy → Kai: "ควรมี animation เยอะกว่านี้"                        │
│                                                                 │
│  Nami and Kai detect conflict                                  │
│                                                                 │
│  Nami → Kai: "Conflict detected on design direction            │
│              John: minimal                                      │
│              Amy: animated                                      │
│              Need escalation?"                                  │
│                                                                 │
│  Kai → Nami: "Agree. Escalate to decision maker (Modz)"        │
│                                                                 │
│  Nami + Kai → Robin: ConflictAlert                             │
│  {                                                              │
│    type: "design-direction",                                    │
│    positions: [                                                 │
│      { buddy: "Nami", position: "Minimal design" },            │
│      { buddy: "Kai", position: "Animation-rich design" }       │
│    ],                                                           │
│    context: "Landing page for product launch"                  │
│  }                                                              │
│                                                                 │
│  Robin → Modz: "John กับ Amy เห็นต่างกันเรื่อง design           │
│                                                                 │
│                John: Minimal, clean                             │
│                → Pros: Fast load, professional                  │
│                → Cons: Less engaging                            │
│                                                                 │
│                Amy: Animation-rich                              │
│                → Pros: Engaging, memorable                      │
│                → Cons: Slower, more work                        │
│                                                                 │
│                Context: Landing page for conversion             │
│                                                                 │
│                เธอตัดสินใจได้เลย หรือให้ทั้งคู่มา discuss?"        │
│                                                                 │
│  Modz: "Minimal เป็นหลัก แต่มี subtle animation ได้              │
│         บอกทั้งคู่ว่า both ideas good แต่ speed สำคัญ"           │
│                                                                 │
│  Robin: Decision logged                                         │
│  Robin → Nami: "Decision: Minimal + subtle animation           │
│                Rationale: Speed important for conversion        │
│                Modz says both ideas were good"                  │
│                                                                 │
│  Robin → Kai: [Same]                                           │
│                                                                 │
│  Nami → John: "ได้คำตอบแล้ว: Minimal เป็นหลัก                    │
│               เพิ่ม subtle animation ได้                         │
│               Modz ชอบ idea เธอนะ"                               │
│                                                                 │
│  Kai → Amy: "Decision มาแล้ว: Minimal + subtle animation       │
│             เธอได้ใส่ animation บ้าง                             │
│             Modz appreciate input เธอ"                          │
│                                                                 │
│  [Both feel heard, conflict resolved, work continues]          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
